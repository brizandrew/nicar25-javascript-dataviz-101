# JavaScript Data Visualization 101

Interested in building data visualizations for the web? This session introduces the JavaScript foundations behind browser-based charts. We’ll walk through core programming techniques in JavaScript like: variables, data types, built-in functions, fetching external data, and using the Document Object Model (DOM) to change a webpage. These foundations will be used to create a basic example of a visualization.

This is a foundations class for journalists who are new to JavaScript or just need a refresher on some of the fundamentals. You do not need prior HTML/CSS experience, but HTML/CSS are not taught in this session. We also won’t spend much time building full visualizations here; this class is designed to prepare you for Data Visualization 102, where you’ll build a more complete visual project.

Attendees should bring a laptop (no tablets).
